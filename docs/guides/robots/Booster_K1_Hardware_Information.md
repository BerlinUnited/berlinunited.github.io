# Booster K1 Hardware Information

## Power Connectors
- male connector (on robot): **Prolanv PDC0120F0001**
- female connector (on battery): **Prolanv PDC0210F0001**
- battery charging connector: **hollow connector 5.5mm** 

## Digital Connectors
- network connector: **RJ45**
- USB type A 3.?
- USB type C
- mini display port
